# Transportation & Logistics Machine Learning Project  

## Overview  

This repository contains the implementation of core **Machine Learning concepts** applied to a real-world **Transportation & Logistics Tracking dataset**.  

The project is developed as part of the MSc Data Science – Machine Learning (MDS504-3) coursework and demonstrates practical implementation of statistical analysis, regression modeling, and dimensionality reduction techniques.

This repository will be continuously expanded to cover the complete Machine Learning syllabus.

---

## Academic Context  

**Course:** Machine Learning (MDS504-3)  
**Program:** MSc Data Science (AY 2025–2026)  
**Domain:** Transportation & Logistics Analytics  

---

## Dataset Description  

The dataset contains transportation and logistics tracking information such as:

- Delivery time  
- Shipment distance  
- Transportation cost  
- Shipment weight  
- Delay status  
- Route-related attributes  

The dataset is used to explore statistical properties, build predictive models, and perform dimensionality reduction techniques.
source:kaggle

---

## Exercises Implemented (Phase 1)

### Data Exploration using Parametric Methods  

Performed statistical analysis including:

- Mean, Median, Variance, Standard Deviation  
- Correlation analysis  
- Distribution visualization  
- Outlier detection  
- Statistical interpretation of logistics variables  

**Objective:**  
To understand data distribution and relationships between delivery-related variables.

---

### Regression Analysis  

Built predictive regression models to:

- Predict delivery time  
- Analyze relationship between distance and cost  
- Evaluate model performance using R², MAE, MSE  

**Techniques Used:**

- Simple Linear Regression  
- Multiple Linear Regression  
- Model evaluation and residual analysis  

**Objective:**  
To understand supervised learning and predictive modeling in logistics.

---

### Principal Component Analysis (PCA)  

Performed dimensionality reduction to:

- Reduce feature space  
- Identify major variance-driving components  
- Visualize reduced data structure  

**Objective:**  
To understand dimensionality reduction and feature transformation techniques.
